About / Disclaimer

This site is an unofficial fan-made checklist to help players plan and track Hollow Knight: Silksong completion. It is not affiliated with, sponsored, or endorsed by Team Cherry.

The tool focuses on clarity, speed, and privacy: no login, no intrusive tracking, and data stored only in your browser.

Goals

  • Provide a simple structured view of collectible / progression elements.
  • Allow quick progress marking with persistent local storage.
  • Enable bilingual experience (EN / ES) with room for future languages.
  • Offer a stable schema so early planning does not break later.

Data Status

Some entries are provisional until official information is released. These will be refined as reliable sources or direct game data become available. Historical changes are tracked via version control for transparency.

How Progress Is Stored

Your completion state is stored in localStorage only. Clearing site data or using another browser/device will reset it. A future export/import feature is planned so you can sync manually.

Contributing

Found an error, mistranslation, or want to suggest a new category? Open an issue or pull request on the project repository. Contributions that add verified information, improve clarity, or enhance accessibility are welcome.
